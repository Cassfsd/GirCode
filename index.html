HTML

Es un lenguaje de Marcado de Hipertexto, aunque hay opiniones variadas respecto a llamarlo lenguaje, es lo que permite darle la estructura a la página.
Es como la estructura base de una casa, es el esqueleto de la página.
HTML se maneja con etiquetas, que describen que tipo de elemento se va a crear y esas etiquetas pueden contener atributos, que van a describir el comportamiento de esa etiqueta.
Para iniciar una etiqueta escribimos el angular "<" allí elegimos o ecribimos la etiqueta que vayamos a utilizar y la cerramos con />, hay que tener en cuenta que hay etiquetas que para funcionar necesitan de un cierre y otras son de autocierre y no necesitan de /> para funcionar.

Elementos o partes de una página

Body: es la sección donde se visualiza todo y contiene el header, el main, aside y footer.

Header: puede estar o no dependiendo de la página y suele contener el logo y la barra de navegación.

Logo y Nav bar: el nav bar o barra de navegación contiene todos los enlaces para navegar dentro de la página.

Main: es la sección principal del contenido mostrado en la página.

Aside: es una sección aparte del main, que contiene información de la página pero que no esta directamente relacionada a su contenido principal.

Footer: es el pie de página. Suele ser utilizado para colocar las redes sociales de la página, pero puede contener otros elementos también.

Crear una carpeta y archivo en Visual code

Una vez descargado el VS, vamos a crear en nuestro escritorio una carpeta con el nombre que vayamos a utilizar para el proyecto.

Vamos a abrir nuestro VS y en la parte superior izquierda donde dice archivo/file, vamos a crear un nuevo archivo/new file, donde dice text file, seleccionamos y allí nos dará para elegir el lenguaje que vamos a utilizar y elegimos HTML.

Para crear las primeras líneas básicas vamos a colocar !, damos enter y allí nos crea automaticamente las primeras líneas de código.

A partir de esto, podemos comenzar a diseñar nuestra página web, lo ideal sería tenerla maquetada previamente con alguna herramienta como Figma, Balsamiq, MySketch o alguna de tu preferencia. Para guardar este archivo, podemos ir a archivo/file en la parte superior izquierda o con CTRL+S y seleccionamos guardar como, elegimos la carpeta donde vamos a guardar y el nombre del archivo, SIEMPRE debe ser index.html para que el navegador lo tome y en el tipo de archivo, debemos colocar HTML y le damos guardar.

Podemos guardar cada cierto tiempo con CTRL+ S o podemos colocar el guardado automático yendo a archivo/file y marcar la opción de autoguardado/autosave.
